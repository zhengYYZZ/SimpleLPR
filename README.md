# SimpleLPR
基于深度学习的中国车牌识别

---
## 流程
### 定位车牌
使用目标检测算法定位车牌，如yolo,Mask-RCNN

### 矫正车牌
矫正车牌角度、拉长、缩短等，消除噪声

### 分割字符
将检测出来的车牌分割成单个字符  
`算法`  
- [x] 基于联通区域
- [x] 基于投影直方图
- [ ] 基于滑动窗口
- [ ] 基于过分割
- [ ] 基于TextBox

### 识别字符
通过字符分割，只需识别单个字符
